# fract-ol
Fractal project in 42 graphical branch. Developped in C using miniLibX Library.

## Features:
- Zoom (using keyboard or mouse)
- Color modification
- Translation
- Fractal switch
- Fractal parameters modification using keyboard/mouse

## Available fractals:
- Mandelbrot
- Julia
- N-Julia
- Barnsley Fern
- Burning Ship
- Tricorn
