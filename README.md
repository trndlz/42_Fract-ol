# fract-ol

Fractal project within the 42 graphical branch. Developped in C using MinilibX Library.

### Features:
- Zoom (using keyboard or mouse)
- Color modification
- Translation
- Fractal modification

### Available fractals:
- Mandelbrot
- Julia
- N-Julia
- Barnsley Fern
- Burning Ship
- Tricorn
